# إحصائيات نقرات التواصل

إضافة لـ WordPress تتيح لك إضافة أيقونات هاتف وواتساب مخصصة في مقالاتك وصفحاتك مع عرض إحصائيات دقيقة للنقرات على هذه الأيقونات. تتيح لك الإضافة أيضًا تحديد فترة زمنية لتصفية الإحصائيات وعرض التفاصيل المتعلقة بكل نقرة مثل الرقم والتاريخ.

## المميزات

- عرض أيقونات مخصصة للهاتف والواتساب في مقالات وصفحات WordPress.
- حفظ عدد النقرات على كل أيقونة (هاتف وواتساب).
- عرض إحصائيات النقرات على لوحة التحكم في WordPress مع تفاصيل دقيقة مثل الرقم والتاريخ والوقت.
- إمكانية تصفية الإحصائيات بناءً على التاريخ والنوع (هاتف أو واتساب).
- واجهة مستخدم مريحة لعرض الإحصائيات.
  
## التثبيت

1. قم بتنزيل الإضافة كملف ZIP.
2. من داخل لوحة تحكم WordPress، انتقل إلى "الإضافات" > "أضف جديد" > "رفع إضافة".
3. قم بتحميل ملف ZIP الخاص بالإضافة.
4. بعد التثبيت، قم بتفعيل الإضافة.
5. يمكنك الآن إضافة الأرقام الخاصة بالهاتف والواتساب لكل مقالة عبر المربع الجانبي في محرر المقالات.

## الاستخدام

### إضافة أرقام التواصل:
1. قم بفتح أي مقالة في لوحة التحكم.
2. في المربع الجانبي تحت "Contact Numbers"، قم بإدخال رقم الهاتف ورقم الواتساب.
3. قم بحفظ التغييرات.

### عرض الإحصائيات:
- للعرض الكامل لإحصائيات النقرات، انتقل إلى صفحة "إحصائيات الاتصال" في لوحة التحكم.
- يمكنك تصفية الإحصائيات حسب التاريخ، الرقم، أو عنوان المقالة.

## الإحصائيات:
- يتم تخزين نقرات الهاتف (phone_clicks) ونقرات الواتساب (whatsapp_clicks) بشكل منفصل في قاعدة البيانات.
- يتم عرض جميع النقرات مع التفاصيل مثل الرقم والتاريخ والوقت وعدد النقرات لكل نوع.
